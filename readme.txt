== Description ==

Provides a “Pay on Pickup” option for WooCommerce.

**Note:** This plugin is maintained on GitHub, if you'd like to contribute, submit a pull request here:  
https://github.com/editiontirol/woocommerce-pay-on-pickup/pulls


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the “Plugins” screen in WordPress.
3. Go to WooCommerce->Settings->Checkout->Pay on Checkout to configure the plugin.
